# project07 — v3 고도화 노트 (문화재 AI 해설 앱)

> 2026-06-05 · 단순 해설 → 청중 수준별 AI 도슨트 + TTS 오디오 가이드

## 지능화
- 단일 JSON 호출로 **요약·섹션 해설·흥미로운 사실·용어집·관람팁** 생성
- **청중 수준 적응**(아동/일반/전문) — 같은 문화재를 수준별로 재해설
- 내장 문화재 DB(경복궁·석굴암·훈민정음) 오프라인 폴백

## 인터랙션 / 디자인
- **TTS 오디오 가이드** — 해설 순차 낭독 + 현재 단락 하이라이트
- 용어 풀이 카드, 관람 팁, **관람 도록**(localStorage)

## 콘텐츠
- Info 탭: AI 파이프라인·기술노트·수준별 해설/멀티모달 관람 가이드

## 스택
React 18 · TypeScript · Vite · OpenAI GPT · Web Speech API · localStorage
